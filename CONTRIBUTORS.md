## How to Contribute
1. **Fork** the repository.
2. **Create a branch** for your feature or bug fix.
3. **Commit** your changes with clear messages.
4. **Push** to your branch.
5. **Submit a Pull Request**.

## Guidelines
- Follow the existing code style (Jetpack Compose / Kotlin).
- Ensure all resources are properly qualified (e.g., adaptive icons).
- Test on both handheld and Wear OS devices before submitting.
